# ブランチ管理
## feature/<機能名> ブランチ
```
/your-image-app-repo (feature/image-upload-ui ブランチの例)
├── frontend/
│   ├── app.py                   # UIの変更、画像アップロードロジックの追加
│   ├── requirements.txt         # 新規ライブラリ追加があれば変更
│   └── ...
│
├── .gitignore                   # 必要に応じて新しい一時ファイルを無視する設定を追加
├── README.md                    # その機能に関するメモを一時的に追加することも
└── (その他、必要な変更ファイルのみ)
```
